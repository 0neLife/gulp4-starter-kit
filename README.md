Requirements
-----------
* [NodeJS](http://nodejs.org/) >= v12.x.
* [npm](https://www.npmjs.com/) v6.x.

Run
-----------
```

$ npm install --save-dev gulp

$ npm install

$ gulp
```

First instalation
-----------
```
$ npm install -g gulp

$ npm install jade --global

$ npm install -save-dev gulp

$ npm install

$ gulp
```
